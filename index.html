Go语言结构

发表于 2021-08-23 分类于 Golang

数据类型

  1. 布尔类型（bool）

    取值为 true 或 false，默认是 false，无法参与数值运算，也无法与其他类型进行转换。

  2. 字符串类型（string）

    使用 UTF-8 编码，默认初始值是空字符串，不是 nil。

  3. 数字类型

    • 整型包括：有符号整型：int8/int16/int32/int64 和无符号整型：uint8/uint16/uint32/uint64，此外也有与平台相关的类型：int/uint/uintptr
    • 浮点型包括 float32 和 float64
    • 复数类型包括：complex64（32位实数和虚数）和 complex128（64位实数和虚数）

    数字类型默认均为 0。

  4. 派生类型

    • 数组类型（array）
    • 结构化类型（struct）
    • 指针类型
    • Channel 类型（chan）
    • 切片类型（slice）
    • Map 类型（map）
    • 函数类型（func）
    • 接口类型（interface）

    其中，array、struct 是值类型，默认为 0，剩余均为引用类型，默认为 nil。
