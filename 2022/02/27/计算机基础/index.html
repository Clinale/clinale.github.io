计算机硬件组成

现代计算机以冯·诺伊曼结构为基础，由中央处理器（CPU）、内存、外存、输入和输出设备组成，如图 1 所示。

  • CPU 指令集分为精简指令集（RISC）和复杂指令集（CISC），ARM 指令集属于 RISC，x86 指令集属于 CISC。CPU 由控制单元（CU）和算术逻辑单元（ALU）组成。CU 是计算机的指挥系统，一般由指令寄存器、指令译码器、时序电路和控制电路组成，基本功能是从内存取指令和执行指令。ALU 则负责完成算术运算和逻辑运算。

  • 内存是易失型存储器，包括动态随机存储（DRAM）、静态随机存储（SRAM）以及只读存储器（ROM），其中，SRAM 一般用于制作缓存，ROM 用来存储 BIOS 或固件程序。程序必须被加载进内存才能被执行。

  • 外存是非易失型存储器，主要包括硬盘（HDD）、固态硬盘（SSD），用于永久性存储文件数据。

  • 输入设备主要包括键盘、鼠标和体感设备等，输出设备包括显卡、打印机、扬声器、蓝牙耳机、投影仪等。

  • 主板负责将 CPU、内存、外存、输入输出设备组织在一起。其中，CPU 通过 北桥 与内存和显存连接在一起，输入输出设备则通过 南桥 与 CPU 进行交互。主板主要提供了以下功能：

    1）设备 I/O 地址与 IRQ 中断请求

    主板负责各个电脑组件之间的通信，通过 I/O 地址识别不同的设备，一般而言，不同的设备不能共享同一个设备 I/O 地址。各个设备可以通过 IRQ 中断请求 来告知 CPU 该设备的当前状态，以方便 CPU 进行工作任务的分配。

    2）CMOS 与 BIOS

    CMOS 主要功能为 记录主板上面的重要参数，包括系统时间、CPU 电压与频率、各项设备的 I/O 地址与 IRQ 等，由于这些数据的记录要用电，因此主板上面有一块电池。BIOS 是写入到主板上某一块 flash 或 EEPROM 的程序，它可以在计算机启动的时候执行，以 加载或修改 CMOS 当中的参数，并尝试调用存储设备中的引导程序，进一步进入操作系统当中。

    3）连接外置设备的接口

    主板上面包含各种连接外置设备的接口，包括：

    • PS/2 接口：原本是常见的键盘与鼠标的接口，逐渐被 USB 接口取代；

    • USB 接口：分为 USB2.0 和 USB3.0

    • RJ-45 网络头：负责连接网线

    • HDMI：负责连接显示屏幕，可同时传输声音与图像。

操作系统

主流操作系统分为 Windows 操作系统、Linux 操作系统、Macos 操作系统、Android 操作系统与苹果操作系统等，主要提供了如下功能：

  • 系统调用接口

    对硬件资源进行抽象，向上层应用程序提供一个统一的编程接口

  • 进程管理

    支持多任务处理

  • 内存管理

    实现内存资源的分配与回收，支持虚拟内存，当内存不足时通过硬盘交换分区实现进程切换

  • 文件系统管理

    实现外存资源的分配与回收

  • 驱动管理

    通过驱动程序对硬件进行管理，事先将驱动程序编译成模块，并通过操作系统提供的可加载模块功能加载到操作系统内核空间中

Linux 操作系统中继承了 Minix 操作系统的设计哲学，即所有的程序或系统设备都是文件。各硬件设备在 Linux 中的文件名如表 1 所示。

磁盘分区

磁盘主要由碟片、磁头、机械手臂与主轴马达所组成，碟片用于存储数据，磁头用于写入和读取数据。碟片上面又细分为磁道（Track）和扇区（Sector），所有碟片的同一个磁道被称为柱面（Cylinder）。扇区的物理大小分为 512 字节和 4K 字节两种，典型的碟片示意图如图 2 所示。

物理硬盘无法直接被使用，只有在该硬盘上划分出分区后，才能够使用。其实仅仅划分出分区后，这块硬盘仍然是不能被使用的，需要在划分好的分区上创建文件系统，比如 FAT 或 NTFS，之后才能够被使用。

常见的磁盘分区格式分为 MBR（Master Boot Record）格式和 GPT（GUID partition table）格式两种。在 MBR 格式中，扇区位置使用（柱面号 C、磁头 H、扇区号 S）来表示，最大支持 4TB 的硬盘空间。
